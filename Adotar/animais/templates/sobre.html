Olá meu nome é Matheus Ramos Cardoso

Estou cursando o 3ºAno de Análise e Desenvolvimentos de Sistemas no Instituto Federal do Paraná - Campus Paranavaí e desenvolvendo meu TCC.

Meu TCC consiste no desenvolvimento de um software WEB/MOBILE, para o gerenciamento de viagens com o veículo oficial do campus.
Esse software vai conter alguns cadastros como: Cadastro de servidores, cadastro de dependentes, cadastro de responsaveis, cadastro de veiculo e cadastro de solicitação de viagem.

OBJETIVOS ESPECÍFICOS

  • Gerir os veículos oficiais do campus;
  • Gerir usuário e perfil de acesso, separados por níveis de acesso;
  • Gerir datas disponíveis para reserva;
  • Gerir reserva/solicitação de agendamento de viagens;
  • Gerir o abastecimento e consumo dos veículos;
  • Desenvolver em duas plataformas, uma web para desktops e um aplicativo para dispositivos mobile;
  • Desenvolver uma versão mobile dedicado ao motorista, para inserir os dados de viagens como: quilometragem de saída e de chegada, quantidade de combustível no início e no fim da viagem e demais dados necessários;
  • Desenvolver uma versão mobile dedicado ao usuário para verificar as viagens programadas;
  • Desenvolver um alerta de viagens no aplicativo;
  • Enviar e-mails para os envolvidos notificando todo o processo;
  • Emitir relatório de viagem contendo: nome do motorista, passageiros e local da viagem;
  • Emitir relatório de consumo e quilometragem do veículo.
